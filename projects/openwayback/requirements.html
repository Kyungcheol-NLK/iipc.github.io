Runtime Requirements

JAVA

Tested working with SUN v1.6. It is highly recommended you use the latest version available for your operating system.

Tomcat

Tested working with Apache Tomcat 5.5, and 6.0.

Build from src Requirements

Maven

If you want to build distributions and the website, you'll need Maven 2.
